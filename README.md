###使用手順

#以下のコマンドをPowerShell上で実行してps1ファイルが実行できるようにする
```
Set-ExecutionPolicy RemoteSigned
```
#SteamでPogostuckを右クリック、プロパティを開き起動オプションに以下を入力
```
-diag
```
#狙いたいシードを各行で記載したファイルを作成する
想定は以下のように最低2行以上のシードを記載したファイル
'''
24275
25208
'''

#ps1ファイル3行目、"$seedsPath = "C:\Users\kimuk\Documents\pogo\seeds.txt"のパスを上記で作成したテキストファイルのパスに変更する

#ps1ファイルを起動するとlootSpeedrunの現在のシード値が1秒毎に表示され、目標シードまでの待機時間が表示される

#ポゴのリセットキーとLiveSplitのグローバルホットキーを対応させておくことで、リセットとタイマー起動を同時に行う

#タイマーが待機時間と一致するタイミングを狙ってリセットをかける。狙ったシードを引ければ次のシードまだえの待機時間が0.00と表示される
